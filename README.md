# Rungle

A retro-style space shooter with modern neon aesthetics.

## How to Play
1. Open `index.html` in your web browser.
2. Use **Arrow Keys** or **WASD** to move your ship.
3. Press **Space** or **Click** to shoot.
4. Destroy incoming enemies to score points.
5. Avoid getting hit!

## Features
- Dynamic difficulty scaling
- Particle effects
- High score tracking (per session)
- Keyboard and Mouse support
